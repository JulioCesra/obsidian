É um objeto contêiner usado para representar a presença ou ausência de um valor

Métodos:
	ofNullable -> cria um optional que pode ser nulo ou não.
	isPresent -> verifica se há um valor
	orElse -> retorna o valor contido ou um valor padrão se estiver vazio
	orElseThrow -> Lança uma exceção se o valor não estiver presente
