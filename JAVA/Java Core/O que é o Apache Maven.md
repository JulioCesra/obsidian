É uma ferramenta de gerenciamento de projetos, utilizando o arquivo pom.xml (Project Object Model) para gerenciar dependências, estruturas diretórios, compilar, testar e empacotar. 

FLUXO DE FUNCIONAMENTO DO APACHE MAVEN

arquivo pom.xml contendo as dependências do projeto -> maven busca as dependências e baixa automaticamente para a máquina local -> maven estrutura o projeto em main e test -> realiza o processo de limpar, compilar, testar e gerar o arquivo final da aplicação.
